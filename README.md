# Global_2024sink

Key codes for "Dramatic increase in ecosystem respiration causes record-breaking atmospheric CO2 growth rate in 2024". It contains the codes for processing data, fitting the relationship between TER and airtemperature, TWS and GPP, and drawing the maps. All the codes can be opened with MATLAB 2024a software.
